开发分支
***

## mjyy-web-old

### 仓库说明

该仓库复刻于[MJYY_Web](https://github.com/MJYY-MC/MJYY_Web)仓库

### 分支说明

- `mjyy-web-old/old`: 该仓库的默认分支，可合并至`MJYY_Web/old`，接受来自`mjyy-web-old/dev`的合并。
- `mjyy-web-old/dev`: 该仓库的开发分支，用于维护，持续合并至`mjyy-web-old/old`，**不可**合并至`MJYY_Web/dev`。
- `mjyy-web-old/main`: 不对该分支进行更改，**不可**合并至`MJYY_Web/main`，可接受来自`MJYY_Web/main`的合并。
- `mjyy-web-old/gh-pages`: 属于该仓库的构建输出分支，**不可**合并至`MJYY_Web/gh-pages`
